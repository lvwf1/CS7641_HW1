The steps to run the Python Code for windows:
1. Install PyCharm Professional lastest version
2. Import the unzipped folder as new project into PyCharm 
3. Install pandas, numpy, matplotlib, sklearn as project interpreter, make sure all packages are imported and installed, wait until loading is completed
4. Run ".py" file
5. Comment/Uncomment line #121, #122, #123 for decisiontree.py, knn.py, neuralnetwork.py, svm.py, boost.py, line #19, #20 for classifiercomparison.py to switch another dataset to run machine learning method against
